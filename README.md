# PizzaAppJSF
PizzaApp web application developed using JSF technology.

This web application provides below basic functionalities to the customer.
1) Display all the available restaurant.
2) Customer can select a restaurant from the available list of restaurants.
3) Display available menu for the selected restaurant.
4) Customer can select N number of items from the menu list.
5) Display selected menu items along with respective price and total amount to pay.
6) Final confirmation screen will show the selected items , total amount to pay and the order id.

# Jars needed to run this application:
1) json-simple-1.1.1.jar
2) myfaces-api-2.1.5.jar
3) myfaces-impl-2.1.5.jar
4) jsf-facelets-1.1.14.jar
5) jstl-1.2.jar

# pre-requisite:
* Tomcat server.
